# Football Ticket Booking System

## Overview

This project is a Football Ticket Booking System database created using PostgreSQL.

The system manages:

- Users
- Matches
- Bookings

## Tables

- Users
- Matches
- Bookings

## Relationships

- One User → Many Bookings
- One Match → Many Bookings

## Features

- Primary Keys
- Foreign Keys
- SQL Queries
- JOIN Operations
- Subqueries

## Tools Used

- PostgreSQL
- pgAdmin 4
- DrawSQL
- GitHub
